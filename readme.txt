== Installation ==

= Automatic installation =

1. Go to Plugins > Add New > Upload and select the anhlinh-contact-list.zip file from your hard disk
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to menu `AL Contact List` menu in WordPress for custom setting.

= Manual installation =

1. Upload the plugin folder to the `/wp-content/plugins/` directory via ftp
2. Activate the plugin through the 'Plugins' menu in WordPress
